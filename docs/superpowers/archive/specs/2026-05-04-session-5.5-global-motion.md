# Spec — Session 5.5: Global enter animation

Inherits master plan motion grammar (Session 1). Implementation deferred until now per Session 5 design review finding.

## Goal

Build the one promised enter animation as a reusable primitive and apply it to all already-shipped sections (Hero copy column, Trust bar, Problem). Bake into `<Section>` so all future sections inherit it for free.

## Animation spec (locked)

- `opacity: 0 → 1`
- `transform: translateY(8px) → translateY(0)`
- Duration: `400ms`
- Easing: `cubic-bezier(0.16, 1, 0.3, 1)`
- Trigger: IntersectionObserver, `rootMargin: '0px 0px 50px 0px'` (fire 50px before element enters viewport)
- Once: unobserve after first intersection — never re-fire
- Reduced motion: respect `prefers-reduced-motion: reduce` — skip the transform, render visible immediately

## Decisions

### D1 — `<Reveal>` client component, not a hook
A wrapper component (`<Reveal>{children}</Reveal>`) is more ergonomic than a hook (`const ref = useReveal()`). Hook would require every consumer to wire ref + state + className. Wrapper is one import, zero plumbing.

### D2 — Bake into `<Section>` primitive
`<Section>` wraps its children in `<Reveal>` internally so all future sections (Services Session 6 onward) inherit the animation without additional code. Section stays server-rendered; Reveal is the client boundary.

### D3 — Hero gets explicit `<Reveal>` wrap
Hero doesn't use `<Section>` (it's a full-bleed split). Wrap the left copy column in `<Reveal>` directly. Orb (right column) stays unanimated — it has its own Three.js motion and doesn't need an entry fade.

### D4 — Trust bar gets explicit `<Reveal>` wrap
Trust bar doesn't use `<Section>`. Wrap the bar contents in `<Reveal>`. The marquee animation continues to run independently.

### D5 — Above-fold behavior
For Hero (above fold on landing), IntersectionObserver fires immediately on mount because the element is already in viewport. This produces a 400ms fade-in on initial page load. Acceptable and standard. If it reads as gratuitous in practice, easy to remove the wrap from Hero alone.
